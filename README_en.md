# 🎮 AI 五子棋

> An intelligent Gomoku (Five in a Row) game with multiple AI engines

[中文](./README.md) | English

---

## ✨ Features

### 🎯 Game Modes
- **Player vs AI** - Challenge the AI one-on-one
- **AI Battle** - Watch two AIs compete against each other
- **Difficulty Levels** - Easy / Medium / Hard
- **Board Sizes** - 13×13, 15×15, 19×19

### 🤖 AI System
- **Multi-Engine Collaboration** - 11 professional AI modules working together
- **Pattern Recognition** - Intelligent board pattern and threat detection
- **Attack/Defense Strategy** - Dynamic strategy adjustment
- **VCT/VCF** - Continuous attack and defense detection
- **Memory System** - Learn from historical games

### 🎨 Visual Effects
- **Refined Rendering Mode** - 3D pieces, wood grain board
- **Theme Switching** - Dark/Light mode
- **Dynamic Island Animation** - Beautiful music player animations
- **Smooth Animations** - Piece placement, victory, and entry animations

### 📱 User Experience
- **Responsive Design** - Desktop and mobile optimized
- **Sound System** - Rich audio feedback
- **Haptic Feedback** - Mobile vibration support
- **Keyboard Shortcuts** - Ctrl+Z to undo

## 🚀 Quick Start

### Online Demo
Open `index.html` directly in your browser

### Local Development
```bash
# Clone the project
git clone https://github.com/yourusername/gomoku-ai.git
cd gomoku-ai

# Start a local server
python -m http.server 8000
# or
npx serve .

# Open http://localhost:8000 in your browser
```
